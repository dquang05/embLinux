# C/C++ Learning Journey: System Programming & Concurrency

This repository contains my source code, exercises, experiments, and notes while studying low-level Linux programming and modern C++ concurrency.

The learning path is primarily based on two books:

* **Linux System Programming (LSP)** by Robert Love
* **C++ Concurrency in Action** by Anthony Williams

The goal is to build a solid foundation in:

* Linux system calls and operating system interfaces
* Process and thread management
* Modern C++ programming (C++11/14/17/20)
* Concurrent and parallel programming
* Embedded Linux development
