# TLPI Custom Library

This directory contains the custom utility functions and header files provided in *The Linux Programming Interface* (TLPI) by Michael Kerrisk.

## Purpose

The primary purpose of this library is to **standardize and simplify the source code for learning**. 

By providing an umbrella header (`tlpi_hdr.h`), standardized error-handling routines (`error_functions.c`), and safe numeric parsing functions (`get_num.c`), this library eliminates repetitive boilerplate code. This allows readers to focus entirely on the core Linux system programming concepts demonstrated in each exercise without being distracted by routine setup and error-checking mechanics.
